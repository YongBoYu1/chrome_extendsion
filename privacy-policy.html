Privacy Policy

Last updated: March 27, 2024

Overview

This privacy policy describes how our Chrome extension ("Web Summarizer") collects, uses, and protects your information when you use our service.

Information We Collect

Our extension collects the following types of information:

  • Website Content: We temporarily access the content of web pages you choose to summarize.
  • Web History: We access the URL of the current page when you request a summary.
  • User Activity: We collect basic usage data when you interact with the extension (such as when you click to generate a summary).

How We Use Your Information

The information we collect is used solely for:

  • Generating summaries of web content you request
  • Improving the extension's functionality
  • Troubleshooting technical issues

Data Protection

We take the following measures to protect your data:

  • All data transmission is encrypted using HTTPS
  • We do not store any personal information permanently
  • Website content is processed temporarily and immediately discarded

Third-Party Services

Our extension uses the following third-party services:

  • Our secure backend API for processing summaries

These services are used solely for providing the core summarization functionality.

Data Sharing and Transfer

We do not:

  • Sell or transfer user data to third parties
  • Use data for purposes unrelated to the extension's core functionality
  • Use data to determine creditworthiness or for lending purposes
